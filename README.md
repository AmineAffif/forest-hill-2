# Projet Next.js Forest Hill

## Installation

Pour installer les dépendances du projet, exécutez la commande suivante :

```bash
npm install
```

## Démarrage du projet

Pour démarrer le serveur de développement, exécutez :

```bash
npm run dev
```

## Accès au projet

Attention c'est sur le port 3001 :

```
http://localhost:3001
```
